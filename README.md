# Modelado-de-series-temporales-mediante-SARIMAX-y-LSTM-para-Medios-Transaccionales-financieros
Con el objetivo de cumplir con los estándares de la ciencia abierta y permitir la replicación exacta de los análisis de estacionariedad, diagnóstico estocástico, entrenamiento y validación cruzada descritos en esta tesis, se ha estructurado un repositorio estructurado en GitHub.

Este espacio digital contiene:

Los scripts de extracción y limpieza de datos transaccionales para los cuatro canales.
Los cuadernos de optimización de hiperparámetros de la red LSTM (look-back periods, celdas y tasas de aprendizaje).
Las rutinas estadísticas para la ejecución de las pruebas ADF, Ljung-Box, Engle (ARCH) y el Test de Diebold-Mariano.
